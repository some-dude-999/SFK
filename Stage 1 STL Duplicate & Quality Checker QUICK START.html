STL Duplicate & Quality Checker - Quick Start

What This Tool Does

Finds duplicate STL files and lets you mark bad ones for deletion. Saves time when cleaning up large 3D model collections.

How to Use

1. Upload Files

Drag and drop your STL files onto the upload area, or click to select them.

2. Review Results

Tool automatically shows:

  • Total Files - Number uploaded
  • Duplicates - Files with same size or similar names
  • Bad Files - Files you marked for deletion

Color codes:

  • Yellow - Duplicate (delete candidate)
  • Green - Safe duplicate (keep this one)
  • Red - Bad file (marked by you)

3. Inspect Models (Optional)

Click any thumbnail to open 3D viewer:

  • Left click + drag = Rotate
  • Right click + drag = Pan
  • Scroll = Zoom
  • Arrow buttons = Browse all models

4. Mark Files for Deletion

5. Export List

Click "Export Deletion CSV" to download list of files to delete from your computer.

Important

  • Always check duplicates in 3D viewer before deleting - same size doesn't always mean same model
  • First file in duplicate group is marked green (keeper), rest are yellow (delete candidates)
  • When in doubt, keep the file
  • Nothing gets deleted automatically - you control everything

STL Duplicate & Quality Checker - Mabilis na Simula

Ano ang Ginagawa ng Tool na Ito

Hinahanap ang mga duplicate na STL files at pinapayagan kang markahan ang mga masama para sa pagtanggal. Nakakatipid ng oras kapag naglilinis ng malalaking koleksyon ng 3D models.

Paano Gamitin

1. Mag-upload ng Files

I-drag at i-drop ang iyong mga STL files sa upload area, o mag-click para pumili.

2. Tingnan ang Resulta

Awtomatikong ipapakita ng tool ang:

  • Total Files - Bilang ng na-upload
  • Duplicates - Mga files na may parehong laki o magkatulad na pangalan
  • Bad Files - Mga files na minarkahan mo para tanggalin

Mga kulay:

  • Dilaw - Duplicate (pwedeng tanggalin)
  • Berde - Safe duplicate (itago ito)
  • Pula - Masama (minarkahan mo)

3. Suriin ang mga Modelo (Opsyonal)

I-click ang kahit anong thumbnail para buksan ang 3D viewer:

  • Left click + drag = Ikitin
  • Right click + drag = Ilipat
  • Scroll = Zoom
  • Arrow buttons = Tingnan lahat ng modelo
  • ‚ùå button = Markahan bilang masama
  • üóëÔ∏è button = Markahan bilang duplicate

4. Markahan ang Files para Tanggalin

Dalawang paraan:

  • I-click ang "Duplicate" button sa table
  • Gamitin ang ‚ùå o üóëÔ∏è buttons sa 3D viewer

5. I-export ang Listahan

I-click ang "Export Deletion CSV" para i-download ang listahan ng mga files na tatanggalin sa iyong computer.

Mahalaga

  • Laging suriin ang duplicates sa 3D viewer bago magtanggal - parehong laki ay hindi laging parehong modelo
  • Unang file sa duplicate group ay naka-mark na berde (itago), ang iba ay dilaw (pwedeng tanggalin)
  • Kung hindi ka sigurado, itago ang file
  • Walang awtomatikong tinatanggal - ikaw ang kontrolado ng lahat
