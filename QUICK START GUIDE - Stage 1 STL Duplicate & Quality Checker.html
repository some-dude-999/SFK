STL Duplicate & Quality Checker

What is this?

A tool to find duplicate STL files and mark bad files. Upload hundreds of 3D models, review them quickly, and export a list of files to delete.

How to use (5 steps)

  1. Upload STL files - Drag & drop or click to select files
  2. Review duplicates - Tool auto-detects duplicates by size/name
    • Yellow = Duplicate (delete candidate)
    • Green = First file (keep)
    • Red = Bad file (you marked it)
  3. Inspect in 3D viewer - Click thumbnail to view model, use arrows to browse
  4. Mark files - Click "Duplicate" button or use viewer buttons (üóëÔ∏è or ‚ùå)
  5. Export CSV - Download list of files to delete from your computer

Quick tips

  • Always check duplicates in 3D viewer before deleting
  • Mouse controls: Left drag = rotate, Right drag = pan, Scroll = zoom
  • Arrow keys navigate between models in viewer
  • When in doubt, keep the file

Ano ito?

Tool para hanapin ang duplicate STL files at markahan ang bad files. Upload ng daan-daang 3D models, review nang mabilis, at export ng listahan ng files na tatanggalin.

Paano gamitin (5 hakbang)

  1. Upload ng STL files - Drag & drop o click para pumili ng files
  2. Review ng duplicates - Automatic na hahanapin ang duplicates base sa size/name
    • Yellow = Duplicate (kandidato para tanggalin)
    • Green = Unang file (itatago)
    • Red = Bad file (minarkahan mo)
  3. Tingnan sa 3D viewer - Click sa thumbnail para tingnan, gamitin ang arrows para mag-browse
  4. Markahan ang files - Click ang "Duplicate" button o gamitin ang viewer buttons (üóëÔ∏è o ‚ùå)
  5. Export ng CSV - Download ang listahan ng files na tatanggalin sa computer

Mabilisang tips

  • Palaging i-check ang duplicates sa 3D viewer bago tanggalin
  • Mouse controls: Left drag = i-rotate, Right drag = i-pan, Scroll = zoom
  • Arrow keys para mag-navigate sa viewer
  • Kung hindi sigurado, itago ang file
